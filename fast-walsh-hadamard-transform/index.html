前面提到了，离散傅里叶变换的多项式卷积是对于每个 kkk 获得 i+j=ki+j=ki+j=k 的 ∑ai×aj\sum a_i \times a_j∑ai​×aj​，而在数论中比较多的狄利克雷卷积是对于每个 kkk 获得 i×j=ki \times j = ki×j=k 的 ∑ai×aj\sum a_i \times a_j∑ai​×aj​。在实际题目中，我们可能会遇到这样的一类卷积：

对于每个 kkk，获得 i⊗j=ki \otimes j = ki⊗j=k 的 ∑ai×aj\sum a_i \times a_j∑ai​×aj​。其中 ⊗\otimes⊗ 是任意按位逻辑运算，如 &,∣,⊕\&, |, \oplus&,∣,⊕ 等。

很显然，我们如果按照最朴素的枚举方法：先枚举 iii，再获得可行的 kkk，复杂度是 O(n2)O(n^2)O(n2) 朝上走的。如果 n≥105n \ge 10^5n≥105 不就完蛋了吗。按照这个架势，我们应该是要找复杂度为 O(nlog⁡n)O(n\log n)O(nlogn)，O(nlog⁡log⁡n)O(n \log \log n)O(nloglogn)，O(nlog⁡2n)O(n \log^2 n)O(nlog2n) 的算法。

很幸运的是，我们可以通过分治来完成这样的一件事情。当然了，由于位运算的值域是 2n2^n2n 级别的，我们也要像 FFT 那样将数列长度补齐。

void FWT(lld X[], int l, int r, int dft = 1)
{
	if (l == r) return;
	int m = (l+r)>>1;
	FWT(X, l, m, dft);
	FWT(X, m+1, r, dft);
	for (int i = 0; i <= m-l; i++)
		X[l+i] += X[m+1+i] * dft;
}

Niuniu likes playing games. He has n piles of stones. The i-th pile has ai stones. He wants to play with his good friend, UinUin. Niuniu can choose some piles out of the n piles. They will play with the chosen piles of stones. UinUin takes the first move. They take turns removing at least one stone from one chosen pile. The player who removes the last stone from the chosen piles wins the game. Niuniu wants to choose the maximum number of piles so that he can make sure he wins the game. Can you help Niuniu choose the piles?

给出 nnn 个数字，记为 a0,a1,a2,...,an−1a_0, a_1, a_2, ..., a_{n-1}a0​,a1​,a2​,...,an−1​，且 ai≤5×105a_i \le 5\times10^5ai​≤5×105，求 max∣A∣,⊕a∈Aa=0max|A|, \oplus_{a\in A} a = 0max∣A∣,⊕a∈A​a=0

不妨记 ⊕i=0n−1ai=sum\oplus_{i=0}^{n-1}a_i = sum⊕i=0n−1​ai​=sum，那么我们构造一个最小的堆，使得其异或和为 sumsumsum 即可。

我们只要记录可以被构造出来的堆值，就可以在第一次这个堆被构造出来时停止。由于 x⊕x=0x \oplus x = 0x⊕x=0，所以我们直接用已经构造出来的集合和原始给出的集合进行逻辑运算卷积。因为不需要记录方案数，所以直接置1即可。

记 Bkn=[(∑i⊕j=kAiBjn−1)>0]B_{k}^n = \left[\left(\sum_{i \oplus j = k} A_{i} B_{j}^{n-1}\right) > 0 \right]Bkn​=[(∑i⊕j=k​Ai​Bjn−1​)>0]，相当于记录了，有 BBB 的堆和 AAA 原来堆，异或后能构造的值的集合。显然这个数字不会超过 2192^{19}219，所以做19次卷积即可。

#include <cstring>
#include <cstdio>
typedef long long lld;
const int MAX = 524287;
lld a[MAX+1], b[MAX+1] = {1};

void FWT(lld X[], int l, int r, int dft)
{
	if (l == r) return;
	int m = (l+r)>>1;
	FWT(X, l, m, dft);
	FWT(X, m+1, r, dft);

	for (int i = 0; i <= m-l; i++)
	{
		lld x = X[l+i], y = X[m+1+i];
		X[l+i] = dft == 1 ? x+y : (x+y)/2;
		X[m+1+i] = dft == 1 ? x-y : (x-y)/2;
	}
}

int main()
{
	int n, tmp, sum = 0; scanf("%d", &n);
	for (int i = 0; i < n; i++)
	{
		scanf("%d", &tmp);
		a[tmp]++;
		sum ^= tmp;
	}

	FWT(a, 0, MAX, 1);
	int max_ans = 0;
	while (!b[sum])
	{
		if (max_ans++ == 19) break;
		FWT(b, 0, MAX, 1);
		for (int j = 0; j <= MAX; j++)
			b[j] = a[j] * b[j];
		FWT(b, 0, MAX, -1);
		for (int j = 0; j <= MAX; j++)
			b[j] = !!b[j];
	}

	printf("%d\n", n - max_ans);
	return 0;
}

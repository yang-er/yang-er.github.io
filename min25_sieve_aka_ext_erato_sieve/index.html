这也是个用于求积性函数前缀和的筛法。

对于积性函数 f(n)f(n)f(n)，满足这样几个要求时可以用min25来做。

  • 当 n∈primen \in primen∈prime 时， f(n)f(n)f(n) 可以使用关于 nnn 的多项式表示。

  • 当 n∈primen \in primen∈prime 时，f(nk)f(n^k)f(nk) 要能够快速计算答案。

这里是预处理的代码。

// euler sieve, spk for sum p^k from p_1 ~ p_pcn
int pri[MAXN], pcn, spk[MAXN];
int m, Sqrt, id1[MAXN], id2[MAXN];
long long gk[MAXN], w[MAXN];

inline long long sum_ik(lld n); // for i=2..n, sum i^k

void init()
{
    m = 0, Sqrt = sqrt(n);

    for (long long L = 1, R; L <= n; L = R+1)
    {
        R = n / (n / L), w[++m] = n / L;
        (w[m] <= Sqrt ? id1[w[m]] : id2[R]) = m;
        gk[m] = sum_ik(w[m]);
    }

    for (int j = 1; j <= pcn; j++) // p_pcn^2 should be greater than n
    {
        for (int i = 1; i <= m && pri[j] <= w[i]/pri[j]; i++)
        {
            int d = (w[i]/pri[j]<=Sqrt) ? id1[w[i]/pri[j]] : id2[n/(w[i]/pri[j])];
            gk[i] -= (spk[j] - spk[j-1]) * (gk[d] - spk[j-1]);
        }
    }
}

其中，id1[i] 表示数论分块中整除值 ⌊n/l⌋=i\lfloor n/l \rfloor = i⌊n/l⌋=i 的块编号，id2[n/i] 表示整除值 ⌊n/l⌋=i\lfloor n / l \rfloor = i⌊n/l⌋=i 的块编号，w[s] 表示数论分块第 sss 个块的整除值。这样比较节省空间。

然后考虑下面那个迭代。首先，wiw_iwi​ 就是 g 函数的第一个参数。

当 pj2>wip_j^2 > w_ipj2​>wi​，也就是第 iii 个块整除值的时候，没有必要继续处理了。内层的 i=1⋯mi=1 \cdots mi=1⋯m 相当于从大往小枚举块，也就不用利用滚动数组什么的记录上一轮 j−1j-1j−1 时的具体值了。

最后，init执行完时，gk[i] 就是第 iii 个整除值 wiw_iwi​ 时，g(wi,pcn)g(w_i, pcn)g(wi​,pcn) 的值了，也就是我们想要的东西了。

由于那个多项式可能会有好几个 pxp^xpx，所以都维护出来吧。

所以这一部分的代码可以写成

long long S(long long x, int y)
{
    if (x <= 1 || pri[y] > x) return 0;
    int d = x <= Sqrt ? id1[x] : id2[n/x];
    long long ans = g(x) - g(pri[y]-1);

    for (int i = y; i <= pcn && pri[i] <= x / pri[i]; i++)
    {
        long long p1 = pri[i], p2 = p1 * p1;
        for (int e = 1; p1 <= x / pri[i]; ++e, p1 = p2, p2 *= pri[i])
            ans += S(x/p1, i+1) * f(i,e) + f(i,e+1);
    }

    return ans;
}

其中 g(n) 是 ∑i=1nf(i)[i∈prime]\sum_{i=1}^n f(i) [i \in prime]∑i=1n​f(i)[i∈prime]，也就是上面式子中的 g(n,∣P∣)g(n,|P|)g(n,∣P∣) 可以从上面预处理的式子中得到。而 f(i,e) 就是 f(pie)f(p_i^e)f(pie​)。
